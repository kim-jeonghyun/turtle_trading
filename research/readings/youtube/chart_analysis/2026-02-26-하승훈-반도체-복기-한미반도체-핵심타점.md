# 오늘 반도체에서 반드시 복기해야 할 한 자리

## 요약

하승훈이 2026년 2월 26일 반도체 섹터 매매를 복기하며, **한미반도체에서 놓친 핵심 매수 타점**을 분석합니다. 한화비전 2박 3일 스윙 매매를 먼저 보여준 뒤, 당일 1등 테마(반도체)의 1등주(한미반도체)가 컵 위드 핸들 패턴을 완성하며 돌파한 자리를 기계적으로 매수했어야 했다고 자기 반성합니다.

**핵심 포인트 3가지:**
1. **1등 테마의 1등주가 매물 소화 후 돌파하는 자리는 기계적으로 매수** — 개인적 호불호와 무관하게 대응해야 함
2. **매물소화 → 변동성 축소 → 거래 급감 → 거래 급증 돌파** = 컵 위드 핸들 패턴의 핵심 시퀀스
3. **"결과론이 아니라 기준의 문제"** — 사전에 세운 전략 기준에 부합하면 실행해야 하며, 놓쳤을 때 반드시 복기해야 함

## 핵심 인사이트

### 1. 주도 테마 & 주도주 판별 프로세스

2월 24일(전전일) → 25일(전일) → 26일(당일)의 흐름:

| 날짜 | 반도체 대형주 | 소부장 종목 | 핵심 판단 |
|------|-------------|-----------|----------|
| 2/24 (전전일) | 강세 | **초강세** (한화비전 상한가급) | 테마 확인 |
| 2/25 (전일) | 삼성전자·SK하이닉스 2%대 상승 | **쉬어가는 흐름** | 매물소화 국면 진입 |
| 2/26 (당일) | 강세 유지 | 한미반도체 **1등주로 부상** | 주도주 교체 확인 |

**판단 기준**: 소부장 종목들이 "약해진 것"이 아니라 "매물소화 국면에 들어간 것"으로 해석 → 재상승 가능성 열림

### 2. 한화비전 매매 복기 (2박 3일 스윙)

#### 매매 타임라인

```
[2/24] 1등 테마 1등주 확인 → 분할 매수
       → 상한가 부근 일부 청산, 잔여 보유
[2/25] 분할 매도 대응
       → 장마감 무렵 3일 이동평균선 지지 확인 → 종가 매수
[2/26] 프리장 상승 → 정규장 하락 → 저항 반복
       → 올라갈 때마다 부딪힘 → 분할 매도
       → 전일 시초가 부근에서 다수 청산
       → 전일 고가 돌파 시 눌림 추가매수 + 돌파 재매수
       → 저항에서 분할매도 마무리
```

#### 핵심 원칙
- **"강한 구간에서는 가격이 아니라 흐름을 봐야 한다"** — 가격이 높아도 1등 테마 1등주의 신고가 흐름이면 매수
- 주도주 흐름이 바뀌는 것을 확인한 뒤 (한화비전 → 한미반도체) 매매 마무리

### 3. 한미반도체 — 놓친 핵심 타점 (**이 영상의 핵심**)

#### 왜 놓쳤는가?

1. 개인적으로 **선호하지 않는 종목**이었음
2. 이미 한화비전을 보유 중이라 **관망 선택**
3. → 여기까지는 이해 가능한 판단

#### 놓친 자리의 구조 (12시 30분경)

```
[장초반] 강한 상승 → 조정 시작
         ↓
[조정]   바닥① → 바닥② → 바닥③ (삼중바닥, W자형)
         ↓
[반등]   아침 고점 근처까지 회복
         ↓
[매물소화] 눌림 → 반등 → 눌림 → 반등 반복
          변동성 축소 + 거래 급감
         ↓
[돌파]   거래 급증하며 돌파 ← ★ 핵심 매수 타점
```

#### 패턴 분석: 컵 위드 핸들 (Cup with Handle)

```
        ┌─── 아침 고가 ──── 핸들(변동성 축소) ──→ 돌파!
       /                    ├──┤ 거래 급감
      /                     └──┘
     │     컵 (조정)                ↑
      \    ↓                      거래 급증
       \  바닥①②③(삼중바닥)
        └─────────────────────────────┘
```

#### 왜 기계적으로 매수해야 했는가?

| 조건 | 충족 여부 |
|------|----------|
| 1등 테마 (반도체) | ✅ |
| 1등주 (한미반도체 — 상승률 1위, 거래대금 최대) | ✅ |
| 매물소화 완료 (변동성 축소 + 거래 급감) | ✅ |
| 대량 거래 동반 돌파 | ✅ |

**4가지 조건이 모두 충족** → "개인적 호불호와 상관없이 기계적으로 대응해야 하는 자리"

### 4. 놓친 후 대응

```
[돌파 후] 급등 → 상한가 근처까지 상승 (쳐다볼 수밖에 없었음)
         ↓
[첫 조정] 상한가 아래에서 부딪히며 조정
         → 1등 테마 1등주의 "첫 번째 조정" → 분할 매수
         ↓
[저항]   상승 후 저항에서 부딪힘 → 분할 매도
         ↓
[재조정] 하락 → 반등 → 하락 (ABC 조정, 삼파)
         → 전저점 부근 하락 진정 확인 → 분할 매수
         ↓
[청산]   전고점 부근 → 분할 청산
```

**교과서적 대응**: 1등 테마 1등주를 눌림에서 분할 매수, 저항에서 분할 매도

### 추가 맥락: 컵 위드 핸들의 3단계 거래량 법칙

William O'Neil이 정립한 컵 위드 핸들 패턴에서 **거래량은 각 단계마다 명확히 다른 행동**을 보입니다. 이것이 퀴즈 Q2, Q7에서 핵심이었던 "핸들 vs 돌파" 구분의 근거입니다.

#### 단계별 거래량 특성

| 단계 | 가격 행동 | 거래량 | 의미 |
|------|----------|--------|------|
| **① 컵 좌측** (하락) | 고점에서 조정 | 감소 추세 | 매도 압력 감소 |
| **② 컵 바닥** | 바닥 다지기 | 낮은 수준 유지 | 매도자 소진 |
| **③ 컵 우측** (반등) | 고점 회복 | 점진적 증가 | 매수세 복귀 |
| **④ 핸들** (축소) | 좁은 범위 횡보 | **급감** ★ | 에너지 축적 (코일링) |
| **⑤ 돌파** | 저항선 상향 돌파 | **급증** ★★ | 기관 매수세 유입 |

#### 핸들의 정량적 기준 (O'Neil 원칙)

- **깊이**: 컵 높이의 1/3 이하 (깊을수록 약한 신호)
- **기간**: 컵 길이의 1/5 ~ 1/4
- **거래량**: 핸들 형성 중 **눈에 띄게 감소**해야 유효
- **변동성 축소 (Volatility Contraction)**: 가격 변동폭이 좁아지며, 이는 기술적 분석에서 가장 신뢰도 높은 신호 중 하나

#### 핵심 개념: 에너지 축적 (Coiling)

> "변동성이 핸들에서 수축할 때, 본질적으로 에너지가 축적(coiling)되는 것이다. 이 에너지가 돌파 시 폭발적으로 방출된다. 컵 위드 핸들이 깔끔하고 높은 모멘텀의 돌파를 만들어내는 이유가 바로 이 때문이다." — TrendSpider

이것이 한미반도체에서 정확히 일어난 일입니다:
- 삼중바닥(컵) → 아침 고점 회복(우측) → **눌림 반복하며 변동성 축소 + 거래 급감(핸들)** → **거래 급증 돌파**

### 돌파 확인: 거래량 기준

신뢰할 수 있는 돌파의 거래량 기준:

| 수준 | 거래량 | 신뢰도 |
|------|--------|--------|
| 최소 | 평균 대비 **40-50% 이상** 증가 | 기본 확인 |
| 이상적 | 평균 대비 **200% 이상** 증가 | 강한 확인 |
| 기관 참여 | 평균 대비 **500-1000%** | 매우 강한 신호 |

**거래량 없는 돌파는 함정일 수 있다** — 거래량 미동반 돌파는 가짜 돌파(false breakout) 가능성이 높으므로 주의.

### 분봉 컵 위드 핸들의 특수성

하승훈이 분석한 한미반도체 패턴은 **분봉(인트라데이) 컵 위드 핸들**입니다. 일봉 패턴과의 차이:

| | 일봉 | 분봉 (인트라데이) |
|--|------|-----------------|
| 컵 형성 기간 | 수주 ~ 수개월 | 수분 ~ 수시간 |
| 핸들 기간 | 수일 ~ 수주 | 수십분 |
| 거래량 확인 | 일별 거래량 | 분별 거래량 |
| 신뢰도 | 높음 | 상대적으로 낮음 (노이즈) |
| 적합 대상 | 스윙 트레이딩 | **1등 테마 1등주에서만** |

분봉 컵 위드 핸들은 노이즈에 취약하지만, **1등 테마의 1등주**라는 맥락이 더해지면 신뢰도가 크게 높아집니다. 하승훈이 "기계적으로 매수해야 하는 자리"라고 강조한 이유입니다.

### 매물소화와 터틀 트레이딩의 교차점

| 하승훈 (분봉 단타) | 터틀 트레이딩 (일봉 시스템) | 공통 원리 |
|-------------------|------------------------|----------|
| 매물소화 확인 | N일간 채널 형성 | 축적 구간 인식 |
| 변동성 축소 + 거래 급감 | ATR 감소 (N값 축소) | 변동성 수축 = 돌파 임박 |
| 거래 급증 돌파 | Donchian Channel 상단 돌파 | 채널 이탈 = 진입 신호 |
| 분할 매수 | 피라미딩 (0.5N 간격) | 추세 확인 후 추가 진입 |
| 저항에서 분할 매도 | 10일/20일 이탈 시 청산 | 규칙 기반 청산 |

**핵심 연결**: 타임프레임과 도구는 다르지만, **"축적 → 수축 → 폭발"**이라는 근본 원리는 동일합니다. 터틀 트레이딩의 Donchian 브레이크아웃도 본질적으로 N일간의 매물소화(채널 내 횡보) 후 돌파를 포착하는 시스템입니다.

### 흔한 실수 4가지

| 실수 | 설명 | 하승훈 영상과의 연결 |
|------|------|-------------------|
| **가짜 돌파 (False Breakout)** | 거래량 미동반 돌파에 진입 | 거래량 급증 확인이 핵심 |
| **깊은 핸들** | 컵의 50% 이상 되돌림 | 한미반도체는 아침고점 부근 얕은 핸들 = 강한 신호 |
| **조급한 진입** | 핸들 완성 전 진입 | 변동성 축소 + 거래 급감 확인 후 돌파 시 진입 |
| **개인적 호불호** | 패턴 충족해도 선호도로 스킵 | ★ 하승훈이 놓친 핵심 이유 |
