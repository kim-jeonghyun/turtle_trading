# YouTube Digest 저장 규칙

## 저장 경로

```
research/readings/youtube/{카테고리}/{YYYY-MM-DD}-{sanitized-title}.md
```

## 카테고리

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| `chart_analysis` | 차트 분석, 기술적 분석, 매매 기법 | 풀백 전략, 캔들 패턴, 지지/저항 |
| `risk_management` | 리스크 관리, 포지션 사이징, 손절 전략 | 켈리 공식, 드로다운 관리, 자금 관리 |
| `trading_psychology` | 매매 심리, 멘탈 관리, 행동 편향 | 리벤지 트레이딩, FOMO, 규율 |
| `market_analysis` | 시장 분석, 매크로, 섹터 분석 | 시장 breadth, 섹터 로테이션 |
| `backtest_strategy` | 백테스트, 전략 검증, 시스템 트레이딩 | 시스템 성과 분석, 파라미터 최적화 |

- 여러 주제에 걸치는 경우 **가장 핵심 주제** 기준으로 분류
- 새 카테고리 필요 시 이 파일에 추가 후 사용

## 파일명 규칙

- 날짜: 영상 업로드 날짜 (불명 시 처리 날짜)
- 제목: 소문자, 공백은 하이픈, 핵심 키워드 위주로 축약
- 예: `2026-02-15-martin-luk-969-return-pullback-strategy.md`

## 카테고리 판단 기준

1. 영상 제목과 내용의 **주된 교훈**이 무엇인가?
2. "이 영상을 다시 찾을 때 어떤 키워드로 찾을까?"를 기준으로 판단
3. 불확실하면 사용자에게 확인
